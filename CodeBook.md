This code book describes all of the data fields in the tidy_data.txt file.


subject - integer - The ID of the test subject (identifies the number of subjects that took part in the study)

range: 1 - 30

activity - string - The types of activities that subjects were performing when the measurements were taken

Walking
Walking Upstairs
Walking Downstairs
Sitting
Standing
Laying

Measurements
These variables are all of the mean values for the corresponding measurement based on the subject and the activity.  These are all of data type numeric.
